# odin-recipes

## Português
Nesse projeto praticarei os conhecimentos sobre GIT, GITHUB e o HTML.
Praticarei:
Criar repositório no GITHUB
Clonar para a máquina Local
Editar arquivos
Criar arquivos (HTML, README)
Enviar para Staging
Commitar para o repositório remoto no GITUB

No HTML:
Criar Imagens
Links Relativos e Absolutos
Praticar atributos do "<img>" e do "<a>"
Praticar os elementos essenciais do HTML "<meta>" & "<title>"
Praticar a criação de listas ordenadas e não ordenadas

## English
In this project, I will practice my knowledge of GIT, GITHUB, and HTML.
I will practice:
Creating a repository on GITHUB
Cloning to the local machine
Editing files
Creating files (HTML, README)
Sending to Staging
Committing to the remote repository on GITUB

In HTML:
Creating images
Relative and absolute links
Practicing the attributes of “<img>” and “<a>”
Practicing the essential elements of HTML “<meta>” & “<title>”
Practicing the creation of ordered and unordered lists
